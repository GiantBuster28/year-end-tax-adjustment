## 2. システム概要

### 2.1 目的

本システムは、従業員の年末調整業務を電子化・自動化し、給与担当者および従業員双方の業務負荷を軽減することを目的とする。

### 2.2 対象業務

- 扶養控除等申告書の収集・管理
- 保険料控除申告書の収集・管理
- 住宅借入金等特別控除申告書の収集・管理
- 年末調整計算（所得税・住民税）
- 源泉徴収票の発行
- 法定調書の作成・提出支援

### 2.3 適用対象・対象外ケースと注意事項

本システムは **主たる給与支払者（当社）における年末調整** のみを対象とする。以下に、従業員の雇用・収入状況別の取り扱いを示す。

#### 2.3.1 雇用形態別の取り扱い

| 雇用形態 | 年末調整対象 | 備考 |
|---------|------------|------|
| 正社員・契約社員（常勤） | ✅ 対象 | 本システムで一括処理 |
| パートタイム・アルバイト | ✅ 対象（条件あり） | 年収103万円以下でも対象。扶養控除等申告書を提出した者に限る |
| 派遣社員 | ❌ 対象外 | 派遣元会社で年末調整。当社は源泉徴収のみ |
| 業務委託・個人事業主 | ❌ 対象外 | 給与所得に該当しないため対象外。本人が確定申告 |

> **パートタイム・アルバイトの注意点**
> - 「扶養控除等申告書」を当社に提出していることが年末調整の前提条件
> - 他社でも勤務している場合、どちらか一方（主たる勤務先）にしか提出できない
> - 当社を主たる勤務先としない場合は **乙欄** 適用となり、年末調整は行われない

#### 2.3.2 副業・副収入がある場合

本システムによる年末調整は当社給与分のみを対象とする。副業・副収入がある従業員は **別途、確定申告が必要** となる。

| 収入の種類 | 年末調整 | 確定申告 | 備考 |
|-----------|---------|---------|------|
| 当社給与のみ | ✅ 本システムで完結 | 原則不要 | — |
| 副業先の給与（他社） | ⚠️ 当社分のみ | **必要**（金額問わず） | 副業先は乙欄徴収。両社の源泉徴収票をもとに確定申告 |
| フリーランス・業務委託収入 | ✅ 当社分のみ | **必要**（副収入20万円超） | 20万円以下は所得税の確定申告不要だが住民税申告が必要な場合あり |
| 不動産所得 | ✅ 当社分のみ | **必要** | 金額に関わらず確定申告が必要 |
| 株式・FX等の譲渡所得 | ✅ 当社分のみ | 特定口座（源泉徴収あり）なら不要 | 一般口座・源泉徴収なし口座は確定申告が必要 |

> **システム上の対応**
> - 申告書入力時に「副業・他所得の有無」を確認する項目を設ける（SCR-011）
> - 副収入がある旨を申告した従業員には確定申告が必要である旨をシステム上で案内する
> - 当社発行の源泉徴収票（SCR-017）は確定申告時に使用するため、必要に応じてダウンロード可能とする

#### 2.3.3 休業中の従業員

| 休業の種類 | 年末調整対象 | 取り扱い |
|-----------|-----------|---------|
| 産前産後休業（産休） | ✅ 対象 | 休業中も雇用契約が継続。当社給与（育児休業給付金は非課税）で年末調整 |
| 育児休業（育休） | ✅ 対象 | 育児休業給付金は非課税のため年末調整の対象外。当社支給給与分のみ処理 |
| 介護休業 | ✅ 対象 | 産休・育休に準じる |
| 傷病休暇・病気療養中 | ✅ 対象 | 会社から給与が支給されていれば対象 |
| 休職中（無給） | ⚠️ 要確認 | 当年中に給与支給が一切ない場合は年末調整不要。傷病手当金（非課税）のみの受給者は対象外 |
| 出向中（給与を出向先が支払） | ❌ 対象外 | 給与支払者が出向先の場合、出向先で年末調整 |
| 退職済み | ❌ 対象外 | 年末時点で在籍していない場合は年末調整不可。本人が確定申告 |

> **休業中従業員のシステム操作**
> - 休業中の従業員もシステムにログインして申告書を入力・提出できる
> - 管理者は従業員ステータス（休業中/育休中 等）を申告書一覧で確認できる（SCR-101）
> - 育児休業給付金・傷病手当金は「収入」欄ではなく「非課税所得」として扱い、入力不要である旨をシステム上でガイドする

#### 2.3.4 年の途中で入退社した場合

| ケース | 取り扱い |
|--------|---------|
| 年の途中入社 | 前職の源泉徴収票を提出してもらい、当社で合算して年末調整 |
| 年の途中退社 | 年末調整は行わない。退職者は自身で確定申告（退職後も源泉徴収票を発行） |
| 当年入社・当年退社 | 同上（確定申告） |

> 前職の源泉徴収票の添付アップロード機能（SCR-015）を活用すること。

## 5. 計算仕様

### 5.1 給与所得控除額の計算

| 給与収入金額 | 給与所得控除額 |
|-------------|---------------|
| 1,625,000円以下 | 550,000円 |
| 1,625,001円〜1,800,000円 | 収入金額 × 40% − 100,000円 |
| 1,800,001円〜3,600,000円 | 収入金額 × 30% + 80,000円 |
| 3,600,001円〜6,600,000円 | 収入金額 × 20% + 440,000円 |
| 6,600,001円〜8,500,000円 | 収入金額 × 10% + 1,100,000円 |
| 8,500,001円以上 | 1,950,000円（上限） |

### 5.2 生命保険料控除額の計算（新契約）

| 年間払込保険料 | 控除額 |
|---------------|--------|
| 20,000円以下 | 払込保険料の全額 |
| 20,001円〜40,000円 | 払込保険料 × 1/2 + 10,000円 |
| 40,001円〜80,000円 | 払込保険料 × 1/4 + 20,000円 |
| 80,001円以上 | 40,000円（上限） |

※ 一般生命保険料控除・介護医療保険料控除・個人年金保険料控除それぞれ上限40,000円、合計上限120,000円

### 5.3 地震保険料控除額の計算

| 区分 | 年間払込保険料 | 控除額 |
|------|---------------|--------|
| 地震保険料 | 50,000円以下 | 払込保険料の全額 |
| 地震保険料 | 50,001円以上 | 50,000円（上限） |
| 旧長期損害保険料 | 5,000円以下 | 払込保険料の全額 |
| 旧長期損害保険料 | 5,001円〜15,000円 | 払込保険料 × 1/2 + 2,500円 |
| 旧長期損害保険料 | 15,001円以上 | 10,000円（上限） |

### 5.4 所得税の税率

| 課税所得金額 | 税率 | 控除額 |
|-------------|------|--------|
| 1,000円〜1,949,000円 | 5% | 0円 |
| 1,950,000円〜3,299,000円 | 10% | 97,500円 |
| 3,300,000円〜6,949,000円 | 20% | 427,500円 |
| 6,950,000円〜8,999,000円 | 23% | 636,000円 |
| 9,000,000円〜17,999,000円 | 33% | 1,536,000円 |
| 18,000,000円〜39,999,000円 | 40% | 2,796,000円 |
| 40,000,000円以上 | 45% | 4,796,000円 |

※ 復興特別所得税：算出所得税額 × 2.1%

### 5.5 扶養区分の自動判定ロジック

扶養親族の区分は、入力された生年月日（`birth_date`）と同居フラグ（`is_living_together`）から年末調整対象年度の12月31日時点の年齢を基に自動判定する。

| 優先順 | 判定条件 | 扶養区分 | 控除額 |
|-------|---------|---------|-------|
| 1 | 年齢 15歳以下 | 年少扶養親族（控除対象外） | 0円 |
| 2 | 年齢 19〜22歳 | 特定扶養親族 | 630,000円 |
| 3 | 年齢 70歳以上 かつ 同居 | 同居老親等 | 580,000円 |
| 4 | 年齢 70歳以上 かつ 別居 | 老人扶養親族 | 480,000円 |
| 5 | 上記以外（16〜18歳・23〜69歳） | 一般扶養親族 | 380,000円 |

> **判定年齢の基準日**：対象年度の12月31日時点。例：2026年度は2026年12月31日時点の満年齢で判定。  
> **配偶者控除・配偶者特別控除**：配偶者の年間所得に応じて適用区分が変わる（所得48万円以下→配偶者控除、48万円超133万円以下→配偶者特別控除）。

### 5.6 障害者控除額の計算

障害者控除は、**本人（employees.disability_type）** および **扶養親族（dependents.disability_type）** の障害者区分に応じて加算する。

| disability_type 値 | 区分名 | 控除額 |
|-------------------|-------|-------|
| `none` | 対象外 | 0円 |
| `general` | 一般障害者 | 270,000円 |
| `special` | 特別障害者 | 400,000円 |
| `cohabitation_special` | 同居特別障害者 | 750,000円 |

> **同居特別障害者**：特別障害者に該当する扶養親族と同居している場合に適用。`disability_type = 'cohabitation_special'` かつ `is_living_together = true` であること。  
> **本人が特別障害者の場合**：`employees.disability_type = 'special'` または `'cohabitation_special'` で400,000円が控除される。

| 本人の障害者区分 | none / general / special / cohabitation_special の4値のみ許容 |
| 扶養親族の障害者区分 | none / general / special / cohabitation_special の4値のみ許容 |
| 扶養区分（表示） | 生年月日・同居フラグから自動判定し画面上にラベル表示（編集不可） |
| 保険料支払額 | 0より大きい正の整数 |
| 住宅ローン残高 | 0より大きい正の整数 |
| 添付書類 | PDF/JPEG/PNG、ファイルサイズ10MB以内 |

### 9.1 年末調整スケジュール

| 時期 | 作業内容 |
|------|---------|
| 10月上旬 | システム設定：対象年度・税率テーブルの更新 |

| 11月1日〜30日 | 従業員による申告書入力・提出期間 |
| 12月1日〜10日 | 給与担当者による申告書審査・承認 |
| 12月中旬 | 年末調整計算実行・確定 |
| 12月給与日 | 給与システムへのデータ連携・過不足精算 |
| 1月中旬 | 源泉徴収票の発行・配布 |
| 1月末日 | 法定調書・給与支払報告書の提出 |

|------|------|---------|
